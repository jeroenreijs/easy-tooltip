Tooltip properties

- data-tip (string | HTML)
- data-tip-position (top | right | bottom | left, default top)
- data-tip-arrow-position (start | center | end, default center)
- data-tip-bg-color (any webcolor, default #000000bd)
- data-tip-text-color (any web color, default #ffffff)
- data-tip-max-width (parent | length | percentage)
- data-animated (boolean)
